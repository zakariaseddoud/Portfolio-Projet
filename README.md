# Tableau de Bord Automatisé - Power BI

## Description du Projet
* Ce projet consiste en un tableau de bord interactif et automatisé développé sur Power BI.
* Il a été réalisé dans le cadre de mon stage en tant qu'étudiant.
* L'objectif principal était de concevoir un outil de pilotage capable d'analyser simultanément des données historiques et des flux en temps réel.
* L'intégralité des données présentées dans ce fichier sont factices afin de respecter le secret professionnel et les accords de confidentialité de mon entreprise d'accueil.
* Les données ont été générées pour reproduire fidèlement la structure, la volumétrie et la complexité des véritables bases de données manipulées durant ma mission.

## Processus et Outils Utilisés
* Extraction et nettoyage réalisés via l'importation de multiples bases de données au format CSV.
* Utilisation avancée de Power Query pour transformer, manipuler, nettoyer et croiser les différentes sources d'informations.
* Modélisation aboutissant à la création d'un modèle de données relationnel fluide et optimisé.

## Fonctionnalités et Visualisations
* Mise en valeur des indicateurs clés à travers des tableaux et graphiques dynamiques.
* Intégration d'un suivi des ventes pour l'analyse des performances commerciales.
* Gestion de l'état des commandes incluant l'identification des annulations.
* Application de thèmes graphiques structurés et configuration des paramètres d'interface pour assurer une lecture claire des indicateurs.

## Comment utiliser ce projet
* Clonez ce dépôt sur votre machine locale.
* Ouvrez le fichier avec Power BI Desktop.
* Naviguez à travers les différents onglets pour explorer les visualisations et le modèle de données sous-jacent.
